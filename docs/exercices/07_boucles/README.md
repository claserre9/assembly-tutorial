# Exercices — Chapitre 07 : Boucles

## Exercice 1 — Somme de 1 à N

Écrivez un programme qui calcule la somme des entiers de 1 à 100 (= 5050) et quitte avec ce résultat comme code de retour (modulo 256 = 186).

## Exercice 2 — Table de multiplication

Écrivez un programme qui affiche la table de multiplication de 7 (de 7×1 à 7×10), une multiplication par ligne.

Format attendu :
```
7 x 1 = 7
7 x 2 = 14
...
7 x 10 = 70
```

## Exercice 3 — Boucle imbriquée

Écrivez un programme qui affiche un triangle d'astérisques :
```
*
**
***
****
*****
```

Utilisez deux boucles imbriquées. La boucle externe contrôle les lignes (1 à 5), la boucle interne imprime les `*`.

## Exercice 4 — Compter les voyelles

Écrivez un programme qui compte le nombre de voyelles (a, e, i, o, u — minuscules seulement) dans la chaîne "Bonjour le monde" et quitte avec ce nombre comme code de retour.
